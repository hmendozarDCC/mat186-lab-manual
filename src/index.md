## ‚ôø Accessibility Commitment
This manual follows **Universal Design for Learning (UDL)** principles. We believe that Data Science should be accessible to everyone. 

* **Colorblind Safe:** Every graph in this manual uses high-contrast palettes (Viridis).
* **Screen Reader Ready:** All images and charts include descriptive Alt-Text.
* **Keyboard Navigation:** You can navigate this entire site without a mouse.
* **Clear Math:** All formulas use LaTeX for perfect clarity on screen readers.

## üöÄ Getting Started



To succeed in this course, follow these three steps:

### 1. Set up your Environment
We will use **R** and **RStudio**. If you are using a DCC lab computer, these are already installed. If you are using your own laptop, please visit the [Setup Guide](setup.md).

### 2. Access the Data
All datasets used in these labs are available on our **Brightspace** course page. Download the `.csv` files before starting each lab.

### 3. Submit your Work
Each lab concludes with a **Lab Task**. You will submit your final Quarto (`.qmd`) files and the rendered PDF/HTML to Brightspace.
